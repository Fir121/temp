## Array

a non-primitive linear data structure that is a collection of elements of the same type

starts with index 0

in java, arrays are classes

arrays are derived from `Object` class

Steps

1. declaration
2. memory allocation
   in java, memory for arrays is dynamically-allocated
3. initialization
